  • Global Permission: the single summary label this app shows for a user.
  • ADMIN: user is admin (is_admin=true), so full access.
  • READ / EDIT / MANAGE / NO_PERMISSIONS: all explicit grants found for that user agree on one value.
  • UNSET: the DB has no explicit experiment/model permission rows for that user (non-admin). It means "no explicit entries found," not necessarily "cannot access anything."
  • MIXED: explicit grants are inconsistent across resources (example: READ on one experiment, MANAGE on another), so there is no single global value.
